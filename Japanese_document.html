Stripe Secure Payment Module for Zen Cart v3.0.0

はじめに

このZen Cart支払いモジュールはStripe安全な支払いトランザクションに使用される。 モジュールの操作に必要な公開鍵と秘密鍵を受け取るには、Stripe社と契約してサインアップしなければならない。 Stripeは135以上の通貨での支払い処理をサポートしている。

現在、Zen Cartバージョン1.5.8以降およびPHP 8.0.0以降をサポートしている。 One-Page Checkout v2.5.5およびZCA Bootstrapテンプレートv3.7.8でも検証済み。

注：

  1. このモジュールによる金銭的損害については保証していないため、テストモード動作を使用してモジュールを確認し、テスト後も正常に動作していることを確認する必要がある。
  2. Stripeのシステムは、USDやEURなどの小数点以下2桁の通貨を100倍してから、Stripeのサーバーに送信する。 Stripeサーバは受信した番号を1/100に変換して管理する。 日本円は小数点がないため、その数字はそのままStripeサーバーに送信される。 このモジュールは自動的に各通貨に変換するが、使用する前に使用するすべての通貨でテストする。
  3. Stripeダッシュボード=>支払い=>すべてのページを開くと、多くの不完全な支払いが見つかる。 Zen cart check out確認ページが表示されるたびに'ストライプサーバがPI_xxxxxxxxxxを作成するため また、Stripeサーバーは支払いごとに2つのPI_xxxxxxxxxxを作成する。 原因はストライプサーバとZencartの仕様にある。
  4. サイトで ワンページチェックアウト(OPC) お前 必須 追加 ストライプ コンマ区切りの支払方法のリスト（ 設定:: 1ページチェックアウト設定::確認が必要な支払い方法 ).

のインストールとアップグレード

v3.0.0より前のバージョンからのアップグレード

v3.0.0以降の支払いモジュールは v3.0.0より前のバージョンと大きく異なる。 コアファイルの上書きやテンプレートファイルの変更が不要（または必要）

v3.0.0より前のバージョンからv3.0.0以降にアップグレードする場合は、まず、それらの以前のバージョンで必要なさまざまなファイルと編集を削除する必要がある。 詳細はGitHub wikiの記事を参照のこと。

重要： APIキーまたはpayment-moduleのtest-modeステータスを変更した後、payment-moduleのデータベーステーブルにあるすべての既存のレコードを消去する必要がある。

  1. 管理者のTools :: Install SQL Patchesツールに移動する。
  2. テキストボックスに次の情報を入力し、「送信」ボタンを押す。

    TRUNCATEストライプ；

支払いモジュールをアップグレードする

前の（インストール）セクションの手順AからCに従う。

Storefrontでのテスト(テストモード)

支払いモジュールが正しくインストールされ、設定されていることを確認するには、次の手順を実行する。

  1. 1つ以上のアイテムをカートに入れて、チェックアウトプロセスを開始する。
  2. を選択Stripe Payment: Credit Cardpayment モジュールを選択し、チェックアウト確認ステップに進む。
  3. Stripeカード入力フォームは以下のように表示される。 フォームが表示されない場合は、間違ったAPIキーを入力している可能性がある。

  4. テスト用のVISAクレジットカード番号を入力 4242 4242 4242 4242、有効期限日（将来）、CVC内のランダムな3桁の数字、および郵便番号（要求された場合）。 「注文の確認」ボタンを押す。
  5. Stripeダッシュボードに移動して、「 支払いと成功」ボタンをクリック。 情報の合計金額、顧客名、および電子メールアドレスを確認する。
  6. Zen Cart adminの顧客:: Orderspageでメールアドレスと注文合計を確認する。 Zen Cartの合計金額、顧客名、電子メールアドレスをStripe情報と比較する。

支払いモジュールを取り外す

管理者の モジュール::支払い をクリックして、 Stripe Payments：クレジットカード モジュールをクリックし、 アンインストール ボタン。

顧客のストライプレコードを削除する

お客様のクレジットカード情報は暗号化されてデータベースに登録されているが、お客様の判断で削除することができる。 顧客は、「マイアカウント」ページのStripeクレジットカード情報削除ボタンで削除することができる。 削除後、この表示は表示されなくなる。
